# 🏀 Michael Jordan Tribute Page

Landing page interactiva dedicada a la carrera de Michael Jordan, con un enfoque en narración visual y animaciones avanzadas.

### 🚀 Tecnologías Utilizadas
* **Tailwind CSS:** Maquetación avanzada y sistemas de grillas complejos.
* **AOS (Animate On Scroll):** Implementación de efectos visuales al hacer scroll.
* **Feather Icons:** Iconografía minimalista.

### 🛠️ Desafíos Técnicos
* **Manipulación del DOM:** Creación de animaciones de revelado de texto mediante JS.
* **Optimización de Recursos:** Uso de carga diferida (lazy loading) para imágenes de alta resolución.
* **UI/UX:** Interfaz diseñada para mantener el engagement del usuario a través de la narrativa.
